
EnvTransfer

Description
===========

Transfer your environment between computers using Yandex disk.

Author
======

* Author: `Yuri Bukatkin`


Installation
============

``` 
1. Create a Python virtual environment and activate it

    $ virtualenv venv
    $ source venv/bin/activate

2. Install package

    $ pip install envtransfer
``` 

Using
=====

  Authorization

``` 
    $ envtransfer auth
```
